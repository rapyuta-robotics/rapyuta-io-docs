## Open the project
- Set env variable for the spec endpoint  
  Eg: `export SPEC_ENDPOINT='https://device-manager.ep.rapyuta.io/v0/spec'`

- Run the node script  
  `node index.js`

- Copy the following files to the folder to be served
  index.html  
  style/main.css  
  style/prism.css  
  script/main.js
